# TabBar 图标说明

## 图标要求

uni-app 的 tabBar 需要以下图标文件（81x81px PNG 格式）：

- `chat.png` - 会话图标（未选中状态，灰色）
- `chat-active.png` - 会话图标（选中状态，蓝色 #1e88e5）
- `settings.png` - 设置图标（未选中状态，灰色）
- `settings-active.png` - 设置图标（选中状态，蓝色 #1e88e5）

## 快速生成图标

可以使用以下在线工具生成图标：
1. [IconFont](https://www.iconfont.cn/) - 搜索"聊天"和"设置"图标
2. [Iconify](https://iconify.design/) - 丰富的图标库
3. [Flaticon](https://www.flaticon.com/) - 免费图标

## 图标建议

- **会话图标**：可以使用消息气泡、聊天框等图标
- **设置图标**：可以使用齿轮、设置等图标

## 临时方案

如果暂时没有图标，可以：
1. 使用简单的文字图标（修改 pages.json 使用文字）
2. 使用在线图标生成工具快速生成
3. 使用设计工具（如 Figma、Sketch）创建简单图标
